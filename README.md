# factordb-rust

Rust binding for [FactorDB](http://factordb.com/)

Includes a library as well as a simple command line app.

## Installation
## Cargo
```
cargo install factordb
```

## Building from source
```
git clone https://github.com/megascrapper/factordb-rust
cd factordb-rust
cargo build
```
